Upload a file exported from your previous software. Supported formats: CSV, Excel (.xlsx/.xls) or PDF

Power Church — Balance Sheet by Fund: In Power Church go to Accounting → Reports → Balance Sheet, export to Excel or CSV.
The file will have columns like: Acct Code | Description | Balance (or Debit | Credit).
Assets (1xxx) → Debit  |  Liabilities (2xxx) → Credit  |  Fund Balance (3xxx) → Credit
Section labels (ASSETS, LIABILITIES, FUND BALANCE) and total rows are automatically skipped.
After uploading you will see the raw file rows and confirm which columns to use — descriptions will be taken exactly from your file.

Power Church — Income & Expense Statement by Fund: In Power Church go to Accounting → Reports → Income & Expense Statement, export to Excel or CSV.
The file will have columns like: Acct Code | Description | Month to Date | Year to Date | Budget | Difference.
Income (4xxx) → Debit  |  Expenses (5xxx) → Credit  |  Net = Excess Income over Expense → Fund Balance (3001)
Section labels (INCOME, EXPENSES) and total rows ("Total Income", "Total Expenses", "Excess Income over Expense") are automatically skipped.
After uploading you will see the raw file rows and confirm columns — the Year to Date column is used as the import amount.
